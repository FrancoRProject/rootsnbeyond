# rootsnbeyond
------------------------------------------
FrancoR Project Records presents Roots'n'Beyond.  Follow the intertwined journeys of Esti and Nesta in this speculative fiction tale blending literature, reggae, dub and science fiction, accompanied by a compilation of fifteen tracks.  (French version only, for now)
